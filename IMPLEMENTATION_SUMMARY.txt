NUEVO ENDPOINT API
-----------------
POST /api/v1/notes/upload-video
  - Acepta: multipart/form-data
  - Parámetros: title (string), folder (string), video (file)
  - Retorna: NoteResponse con video_url

TECNOLOGÍAS UTILIZADAS
---------------------
Frontend:
  • MediaRecorder API - Grabación de video/audio
  • getUserMedia API - Acceso a cámara/micrófono
  • React Hooks - useState, useRef, useEffect

Backend:
  • Google Drive API v3 - Integración con Drive
  • FastAPI - File upload handling
  • aiofiles - Async file operations

FLUJO DE USUARIO
----------------
1. Usuario hace click en "Video Nota"
2. Navegador solicita permisos de cámara/micrófono
3. Usuario inicia grabación
4. Vista previa en tiempo real
5. Usuario detiene grabación
6. Previsualización del video
7. Ingresa título y carpeta
8. Sistema guarda:
   - Localmente en uploads/
   - En Google Drive (Video/Notas)
   - Metadatos en MongoDB
9. Video nota aparece en la lista
10. Usuario puede reproducir directamente
